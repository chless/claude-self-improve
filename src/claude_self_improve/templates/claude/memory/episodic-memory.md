# Episodic Memory — Session Experience Index

This file indexes raw session experiences for pattern retrieval. It sits at the
base of the memory hierarchy: episodic (here) → semantic (MEMORY.md) → procedural (commands/).

## How to Use

1. After each non-trivial session, `/meta-learn` appends an episode entry below
2. When `/meta-situational-learn` needs to find past similar situations, search by tags
3. When `/meta-self-audit` runs, it reads episodes to find patterns for consolidation
4. Episodes older than 20 sessions can be pruned if their lessons have been consolidated into semantic memory

## Episode Format

```
### Episode: [session-date] — [brief description]
- **Tags:** [comma-separated situation tags for retrieval]
- **Situation:** [what was the context]
- **Strategy:** [what approach was chosen]
- **Outcome:** [what happened — success/partial/failure]
- **Key learning:** [one-sentence takeaway]
- **Consolidated:** [yes/no — has this been absorbed into MEMORY.md or a skill?]
```
