# Skill Candidates

Patterns observed during conversations that may become skills after
further evidence. A pattern should appear here before being promoted
to `.claude/commands/`.

## Format

### [Pattern Name]
- **Observed:** YYYY-MM-DD
- **Context:** What task revealed this pattern
- **Pattern:** Brief description of the multi-step approach
- **Evidence:** How many times applied, how well it worked
- **Blocker:** Which criterion is not yet met (reusable/non-trivial/focused/proven)
